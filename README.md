# Sovereign Wolf X Growth Plugin

Claude skills from the [Sovereign Wolf X Growth Guide](https://sovereignwolf.pro).

Includes 8 skills for growing an X account using Claude as your operator:

- **x-post-draft** — turn raw material into a post in your voice
- **x-reply-trigger** — generate reply trigger options for any post
- **x-queue-review** — audit your TweetHunter queue before the week starts
- **x-voice-calibration** — recalibrate Claude's drafting voice to yours
- **x-content-ideas** — generate post ideas when you have nothing in queue
- **x-growth-diagnosis** — diagnose stalled growth or dropping reply rates
- **x-account-audit** — full account audit against the open-source X algorithm
- **x-profile-review** — review your profile for credibility and conversion signals

## Install

```
/plugin marketplace add https://sovereignwolf.pro/claude-plugin/marketplace.json
/plugin install sovereign-wolf-x@sovereign-wolf
```

## Usage

Skills are namespaced under `sovereign-wolf-x:`. Example:

```
/sovereign-wolf-x:x-post-draft
```

## The Guide

These skills are companions to the Sovereign Wolf X Growth Guide — a paid guide that teaches you to run your X account with Claude as the operating system, grounded in the open-source X algorithm.

Details at [sovereignwolf.pro](https://sovereignwolf.pro).
